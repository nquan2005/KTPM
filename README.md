# Unit Test with JUnit – Student Analyzer

## Mô tả
Chương trình phân tích điểm số học sinh và kiểm thử đơn vị bằng JUnit.

## Chức năng
- Đếm số học sinh đạt loại Giỏi (>= 8.0)
- Tính điểm trung bình các điểm hợp lệ (0–10)

## Công cụ sử dụng
- Java
- JUnit 5
- GitHub Issues

## Cách chạy kiểm thử
- Mở project bằng IDE (IntelliJ / Eclipse)
- Chạy file `StudentAnalyzerTest.java`
- Quan sát kết quả test
